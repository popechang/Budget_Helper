# Budget_Helper
# Бюджетный помощник

Авторы: Попов Михаил, Шафранов Александр, Гвардин Аркадий

Консольная программа для учёта расходов за месяц.

## Возможности

- Добавлять расходы (день, сумма, категория)
- Смотреть все расходы по порядку дней
- Узнать сумму трат за любой период
- Найти день с максимальным расходом
- Сортировать категории по сумме трат
- Отменить последний добавленный расход
- Посмотреть гистограмму расходов по дням

Все файлы должны лежать в одной папке.

## Структура проекта

- **main.py** - главный файл с меню и пользовательским интерфейсом
- **data.py** - классы Expense (расход) и TreeNode (узел дерева), функции insert и inorder
- **expenses.py** - добавление, удаление расходов и поиск дня с максимальной суммой
- **prefix_sums.py** - построение префиксных сумм для быстрого подсчёта за период
- **sorting.py** - сортировка категорий методом вставок
- **undo_stack.py** - стек для отмены последнего действия

## Функции меню

```
1. Добавить расход
2. Показать дерево расходов
3. Сумма расходов за период
4. День с максимальным расходом
5. Сортировка категорий по сумме трат
6. Отменить последний добавленный расход
7. Горизонтальная гистограмма расходов по дням
0. Выход
```

## Алгоритмы решений из курса ОП

- Префиксные суммы — быстрый подсчёт за период (O(1))
- Двоичное дерево — хранение расходов по дням
- Сортировка вставками — для категорий
- Стек — для отмены действий
